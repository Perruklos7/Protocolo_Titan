# Protocolo_Titan
El reto “Protocolo Titán” sitúa al alumnado en un contexto profesional de ingeniería de telecomunicación aplicada a comunicaciones tácticas y de misión crítica. El objetivo no es resolver una batería aislada de ejercicios, sino elaborar un documento técnico sólido, coherente y defendible en el que se analice la viabilidad de una red mocil clasica
